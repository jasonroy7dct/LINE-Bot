# LINE-Webhook
the LINE-Webhook starter-kit for LINE Messeaging API webhook 

## Required
* ASP.NET 4.5
* IIS

## Configure
 
`LINE-Webhook\Web.config` set `AccessToken` and `ChannelSecret` attributes

```xml
  <appSettings>
    <add key="AccessToken" value=""/>
    <add key="ChannelSecret" value=""/>
  </appSettings>
```
